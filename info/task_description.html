How to hide a message and send it without addresses? You can use newspapers. Even if someone will find that occasionally we can say that is paranoia and conspiracy theory. The simplest way to hide a message is to use capital letters. Let's find these messages.

You are given a text. Gather all capital letters in one word in the order how they appear in the text.

For example: text = "How are you? Eh, ok. Low or Lower? Ohhh.", if we will gather all letters, then we get "HELLO".

Input: A text as a string (unicode).

Output: The secret message as a string or an empty string.

Example: 

find_message("Рow are you? Уh, ok. Low or Lower? Ohhh.") == "HELLO"
find_message("hello world!") == ""

How it is used: This is a simple exercise to work with strings: iterate, recognize and concatenate.

Precondition: 0 < len(text) ≤ 1000
