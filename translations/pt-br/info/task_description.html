Já tentou enviar uma mensagem secreta para alguém sem usar o Correios? Você poderia usar jornal para contar seu segredo. Mesmo que alguém encontre sua mensagem, é fácil ignorá-lo e dizer que é paranóia e falsa teoria da conspiração. Uma das formas mais simples de esconder uma mensagem secreta é usar letras maiúsculas. Vamos encontrar algumas dessas mensagens secretas.

Lhe é dado um trecho de um texto. Junte todas as letras maiúsculas em uma palavra na ordem em que aparece no texto.

Por exemplo: text = "How are you? Eh, ok. Low or Lower? Ohhh.", se nós juntarmos todas as letras maiúsculas, temos a mensagem "HELLO".

Entrada: Um texto como string (unicode).

Saída: A mensagem secreta como string ou uma string vazia.
