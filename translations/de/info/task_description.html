Schon mal versucht eine geheime Nachricht an jemanden zu senden, ohne die Post zu benutzen? Du könntest Zeitungen nutzen um deine Nachricht zu übermitteln. Selbst wenn jemand Deine Nachricht findet, ist es leicht, ihn abzuwimmeln und ihm Paranoia und einen Hang zu verrückten Verschwörungstheorien zu unterstellen. Einer der einfachsten Wege eine Nachricht zu verstecken sind Großbuchstaben. Lass uns ein paar dieser geheimen Nachrichten finden.

Gegeben wird ein Textabschnitt. Sammle alle Großbuchstaben der Wörter in der Reihenfolge, in der sie im Text erscheinen.

Ein Beispiel:
text = "How are you? Eh, ok. Low or Lower? Ohhh.", wenn man alle Großbuchstaben sammelt, erhält man "HELLO".

Eingabe: Ein Text als String (unicode).

Ausgabe: Die geheime Nachricht als String oder einen leeren String.
